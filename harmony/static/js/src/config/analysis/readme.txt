SYNOPSIS OF PROGRAMMABLE OPTIONS FOR MUSIC ANALYSIS AND ENHARMONIC SPELLING
iDegrees : analysis of scale degrees in minor keys
jDegrees : analysis of scale degrees in major keys
ijIntervals : analysis of intervals in minor and major keys
iChords : analysis of chords in minor keys
jChords : analysis of chords in major keys
hIntervals : analysis and spelling of intervals where key is undetermined
hChords : analysis and spelling of chords where key is undetermined
